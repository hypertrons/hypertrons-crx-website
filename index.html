Install Hypertrons Extension

Please select your browser

  • Recommend Chrome
  • Recommend Edge
  • 360 Safety Explorar
  • 360 Speed Explorar

1.Download

Get the latest release from the 【Download】 entry.

Download the file "hypertrons.zip", and unzip it to a new folder, for example, named as "hypertrons".

2.Manage Extensions

Open the plugin management page for Chrome.

Turn on the 【Developer Mode】 button on the top right corner of this page, and then refresh.

3.Load from file

Click 【Load Unpacked】button on the top left, and select the folder where the unzipped files are stored(e.g. the folder "hypertrons").

Now, the extension can be successfully loaded~ Make sure the switch is on!

1.Download

Get the latest release from the 【Download】 entry.

Download the file "hypertrons.crx".

2.Manage Extensions

Open the plugin management page for 360 Safety Explorar.

Turn on the 【Developer Mode】 button on the top right corner of this page, and then refresh.

3.Load from file

Drag the downloaded file "hypertrons.crx" into this page.

In the pop-up dialog box "Do you want to add Hypertrons?", select 【Add】.
